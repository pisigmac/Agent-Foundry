# ✨ The Magician IA Reader: AI-Powered NLP & Tarot Insights ✨

Welcome to **The Magician IA Reader**! This project presents a unique application combining the power of Artificial Intelligence with the mystique of tarot reading.

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

## Architecture

This module uses a modular pipeline approach with clear separation between data ingestion, processing, and output generation.

## Configuration

Set your API keys in environment variables or a `.env` file before running.
