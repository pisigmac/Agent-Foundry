# 🎯 Tutorial 1: Your First ADK Agent

Welcome to your first step in the Google ADK journey! This tutorial introduces you to the fundamental concept of creating a simple AI agent using Google's Agent Development Kit.

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

## Architecture

This module uses a modular pipeline approach with clear separation between data ingestion, processing, and output generation.

## Configuration

Set your API keys in environment variables or a `.env` file before running.
