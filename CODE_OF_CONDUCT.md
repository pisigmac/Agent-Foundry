# Code of Conduct

## Our Standards

This project is committed to providing a welcoming and productive environment for all contributors.

- Be respectful and constructive in all interactions
- Focus on technical merit and practical outcomes
- Welcome newcomers and help them get started
- Accept constructive criticism gracefully

## Unacceptable Behavior

- Harassment, discrimination, or intimidation of any kind
- Trolling, insulting comments, or personal attacks
- Publishing others' private information without permission
- Any conduct that could reasonably be considered inappropriate

## Enforcement

Instances of unacceptable behavior may be reported by opening an issue or contacting the maintainers directly. All complaints will be reviewed and investigated promptly.

## Scope

This code of conduct applies to all project spaces, including issues, pull requests, discussions, and any public representation of the project.
