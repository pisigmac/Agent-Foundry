## Legend

- **Priority**: P1 = Must have, P2 = Should have, P3 = Nice to have
- **Status**: planned | in-progress | blocked | completed | cancelled
- **Assigned**: unassigned or @username

## How to Contribute

1. Pick an idea from this list
2. Open an issue with the template name and your approach
3. Build it following the existing patterns
4. Submit a PR with working code, tests, and documentation

## Suggesting New Ideas

Open an issue with:
- Template name and category
- Problem it solves
- Target users
- Key features
- Estimated complexity (simple/medium/complex)
