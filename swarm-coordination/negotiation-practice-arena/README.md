# 🎮 AI Negotiation Battle Simulator

Watch two AI agents battle it out in an epic used car negotiation! Built with **Google ADK** for the backend agents and **AG-UI + CopilotKit** for a jaw-dropping reactive frontend.

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

## Architecture

This module uses a modular pipeline approach with clear separation between data ingestion, processing, and output generation.

## Configuration

Set your API keys in environment variables or a `.env` file before running.
