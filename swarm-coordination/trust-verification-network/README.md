# 🤝 Multi-Agent Trust Layer - Secure Agent-to-Agent Communication

Learn how to build a trust layer for multi-agent systems that enables secure delegation, trust scoring, and policy enforcement between AI agents.

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

## Architecture

This module uses a modular pipeline approach with clear separation between data ingestion, processing, and output generation.

## Configuration

Set your API keys in environment variables or a `.env` file before running.
